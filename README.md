# Calculator

A simple graphic calculator built with plain HTML, CSS, and JavaScript.

## Features

- Basic arithmetic: addition, subtraction, multiplication, and division
- Percent, decimal, clear, and delete controls
- Keyboard support for digits, operators, Enter, Backspace, Escape, and percent
- Responsive graphic interface for desktop and mobile browsers

## Run Locally

Open `index.html` in a browser.
